# Test Task: Simple Calculation

Create a simple Rust program that calculates 2+2 and prints the result.

## Requirements
1. Create a file `output/calc.rs` with a main function
2. The program should print "Result: 4"
3. Compile and run the program
4. Save the output to `output/result.txt`

## Acceptance Criteria
- File `output/calc.rs` exists
- File `output/result.txt` contains "Result: 4"
- Build succeeds without errors
